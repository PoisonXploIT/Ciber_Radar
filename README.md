# CIBER-RADAR: Android Security Audit Suite

![Version](https://img.shields.io/badge/version-1.0.0%20STABLE-neon_green) ![Flutter](https://img.shields.io/badge/Built%20with-Flutter-blue) ![License](https://img.shields.io/badge/License-MIT-purple)

**Ciber-Radar** es una suite de auditoría de seguridad y wardriving para Android desarrollada en Flutter. Diseñada con una estética Cyberpunk/Minimalista, permite a investigadores y entusiastas de la ciberseguridad analizar el espectro electromagnético y digital en tiempo real.

## Características Principales

### 1. Wireless Spectrum (WiFi)
* **Escáner en tiempo real:** Detecta SSID, BSSID, RSSI y Frecuencia.
* **Análisis de Seguridad:** Clasificación visual de redes:
    * **Verde:** WPA3/SAE (Seguro).
    * **Ámbar:** WPA2 (Estándar).
    * **Rojo:** WEP/Open (Vulnerable).
* **OUI Lookup:** Identificación de fabricantes mediante base de datos local.
* **Detección de Redes Ocultas:** Identificación de SSIDs vacíos (`<HIDDEN>`).

### 2. Bluetooth Low Energy (BLE)
* **Rastreador de Proximidad:** Monitorización de dispositivos IoT, Wearables y Balizas.
* **Privacidad:** Detección de direcciones MAC aleatorias (Random/Private).
* **Clasificación:** Identificación de tipo de dispositivo y fabricante.

### 3. LAN Recon (Network Scanner)
* **Escáner Híbrido:** Combina **NSD (Native Service Discovery)** y **TCP Port Scan**.
* **Deep Scan:** Analiza 12 puertos críticos (80, 443, 22, 53, 8080, 8008, 9100, etc.) para detectar IoT, Cámaras IP, Impresoras y Routers.
* **Cero Dependencias Obsoletas:** No utiliza ping ICMP (bloqueado en Android modernos), sino sockets nativos.

### 4. Cellular Interceptor (Anti-Stingray)
* **Implementación Nativa (Kotlin):** Acceso directo al API de Telefonía de Android.
* **Detector de Anomalías:** Alerta visual (**DOWNGRADE ATTACK**) si la conexión cae a 2G/GSM, indicando posible presencia de un IMSI Catcher.
* **Monitor de Torre:** Visualización de CID (Cell ID), LAC, Operadora y nivel de señal real en dBm.
* **Soporte Moderno:** Diferenciación correcta entre LTE y 5G (NR).

### 5. Physical Hunter (Magnetómetro)
* **Detección de Hardware:** Uso del sensor magnético para localizar micrófonos, cámaras o rastreadores GPS ocultos en vehículos o paredes.
* **Alertas:** Indicadores visuales basados en intensidad de campo (µT).

### 6. The Auditor (Reporting)
* **Generación de Informes:** Crea un PDF profesional con tablas detalladas de todas las amenazas detectadas.
* **Snapshots:** Captura el estado de todos los sensores en el momento de la auditoría.

## Stack Tecnológico

* **Framework:** Flutter (Dart 3.x).
* **Core Packages:** `wifi_scan`, `flutter_blue_plus`, `geolocator`, `sensors_plus`.
* **Native Channels:** Código Kotlin personalizado para `TelephonyManager` y `NsdManager`.
* **Architecture:** Services-based architecture con Streams para datos en tiempo real.
* **UI:** Material 3 con tema personalizado Dark/Neon.

## Instalación

1.  Clonar el repositorio.
2.  Instalar dependencias:
    ```bash
    flutter pub get
    ```
3.  Conectar dispositivo Android (Modo Desarrollador activo).
4.  Ejecutar:
    ```bash
    flutter run --release
    ```

## Permisos Requeridos

La aplicación solicitará los siguientes permisos en tiempo de ejecución para funcionar:
* `ACCESS_FINE_LOCATION` (Obligatorio por Android para escanear WiFi/Celdas).
* `READ_PHONE_STATE` (Para leer Cell ID).
* `BLUETOOTH_SCAN` / `BLUETOOTH_CONNECT`.
* `NEARBY_WIFI_DEVICES`.
